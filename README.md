## 简介
2016.02 – 2016.05 社区医疗公共服务管理系统，源于毕业设计，基于原生Java Web 做的移动端网站，路由部分和数据库操作部分都是通过反射完成。主要功能是医疗资讯浏览，查看各项体检记录，预约服务医生，消息反馈，查看电子病历，饮食推荐，运动推荐，作息计划，个人信息。
